StaffControlBack
1. Скачать репозиторий
2. Зайти в папку StaffControlBack, открыть решение (StaffControlBack.sln) в Visual Studio
3. В зависимости от типа баз данных на локальной машине, необходимо зайти в проект StaffControl в файл конфигурационной настройки appsettings.json и прописать там строку покдключения к базам данных в строке "DefaultConnection"
4. Аналогично в файле Sturtup.cs в проекте StaffControl проверить настройки провайдера баз данных в строке 28
5. Открыть командную строку (Package Manager Console) и, выбрав проект Binding, произвести миграцию баз данных командой Add-Migration /*Имя миграции*/. 
Так же произведется первоначальное занесение данных в базу, что облегчит тестирование и работу
6. Запустить проект на выполнение (откроется страница браузера с первоначальными данными в формате json)

StaffControllFront
1. Отркыть папку StaffControlFront в npm консоле или Visual Code
2. Произвести установку пакетов node_modules командой npm install
3. Произвести звпуск проекта командой npm run dev
